# Vanilla Invisible Armor Resource Pack

A lightweight Minecraft resource pack that hides all default armor sets visually, allowing your player skin to remain fully visible on multiplayer servers without needing any mods or external loaders.

## About This Project
> **Note on Creation:** When I play minecraft and obtain armor, it always blocks my mc skin. All the fixes online required using curseforge, which I don't know if they work online, so I made this and decided I'll share it for others too. 

## Features
* **100% Vanilla:** Works completely client-side without any mods.
* **Multiplayer Safe:** Fully functional on all multiplayer servers.
* **Modern Compatibility:** Formatted using the latest folder structures for current versions of Minecraft.

## How to Install
1. Download the repository folder (or ZIP and extract, don't put in zip file).
2. Drop it into your `%appdata%\.minecraft\resourcepacks` directory.
3. Activate it in your game's **Options > Resource Packs** menu.
